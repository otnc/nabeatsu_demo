# nabeatsu_demo

数え続ける世界のナベアツbot ([@3_aho_bot](https://twitter.com/3_aho_bot)) に感化され、ローカルで動かすためのデモを作成しました。

`./src/main.ts` がデモ本体で、 `./src/nabeatsu.ts` に判定関数と変換関数があります。

作者の実行環境: Windows 11 + Node.js v24.16.0 (nvm-windows)

デモ動画: https://twitter.com/rin_montblank/status/2076739400122466485

```bash
npm run build
npm start
```
